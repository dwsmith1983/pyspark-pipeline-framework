# pyspark-pipeline-framework

Configuration-driven PySpark pipeline framework with HOCON configuration.

## Installation

```bash
pip install pyspark-pipeline-framework
```

## Features

- Configuration-driven pipeline orchestration
- HOCON-based configuration
- Type-safe component model
- Lifecycle hooks for logging, metrics, and audit trails
- Fault tolerance with retry policies and circuit breakers
- Secrets management integration
- Streaming and batch support
